## About Me

- React와 TypeScript를 중심으로 사용하기 편한 웹 화면을 구현합니다.
- Node.js, Express, PostgreSQL을 활용해 서비스에 필요한 API와 데이터 흐름을 설계합니다.
- 기능 구현에서 끝내지 않고, 실행 방법과 유지보수 흐름까지 정리하려고 합니다.
- 최근에는 분석 서비스, 클론코딩, 팀 프로젝트를 통해 풀스택 개발 경험을 쌓고 있습니다.
